Dariya Helmle — Personal Styling Website

A modern, elegant website for a personal stylist, designed to reflect a premium fashion experience. This project showcases a refined UI, smooth interactions, and a strong brand identity tailored for high-end personal styling services.

🌿 Overview
This website represents the digital presence of Dariya Helmle, a personal stylist offering curated styling services for modern women.

The design focuses on:
Minimalism and elegance
High-end fashion aesthetics
Smooth user experience
Clear service communication
🧩 Features
🎯 Hero Section with strong brand messaging
👗 Services Section with expandable details
🖼 Portfolio (Lookbook) showcasing styling directions
💬 Testimonials from clients
📩 Contact Form with validation and feedback (toast notifications)
📱 Fully responsive design (mobile-first)
✨ Smooth scroll animations and micro-interactions
🛠 Tech Stack
React / Next.js (or React, depending on your setup)
TypeScript
Tailwind CSS (custom design system using OKLCH colors)
Lucide Icons
Custom hooks (e.g., scroll animations)
🎨 Design System

The UI is built around a refined neutral palette using OKLCH color space, ensuring:
Visual consistency
Soft contrast
Premium feel

Typography combines:
Display serif for elegance
Clean sans-serif for readability
